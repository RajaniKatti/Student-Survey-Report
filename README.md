# 🛒 Store Survey Dashboard (Power BI)

## 📌 Project Overview

This project showcases a **Store Survey Dashboard** built using Power BI to analyze customer feedback and store performance.

It provides insights into customer preferences, satisfaction levels, and key trends to support better business decisions.

---

## 🎯 Objectives

* Analyze customer survey responses
* Identify trends and patterns
* Improve store performance using data insights

---

## 📂 Project Files

* `Store_Survey_Report.pbix` → Power BI dashboard
* `images/` → Dashboard screenshots
